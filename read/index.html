# 少有人走的路

# 少有走的路1

自律 方法：推迟满足感，承担责任，忠于事实，保持平衡 其中关于教育孩子自律，一定要身体力行。 解决问题的时机：缺乏耐心和希望问题自行消失。要想解决问题，一定要付出一定的时间和耐心。希望问题自行消失在自身也有一定的体现。譬如婚姻，总希望某一天对方会主动提出离婚，实际上我需要考虑的是这段婚姻是不是自己真心想维护的，若不是，便离开，若是，便用心维护。对待她我也缺乏耐心，总希望她自己可以慢慢变好。对待自己也是，总希望自己可以变得更强大，却不愿多花时间在读书上面。

神经官能症和人格失调症：感觉自己两者皆有。

逃避自由：其实就是放弃自己的力量，导致自己没有选择和决定的机会，也就是不自由。所以这个说法也就是逃避责任带来的痛苦，如果克服了这个痛苦，其实我们是更自由了。不由联想到去年当组长的经历，总是不断的抱怨琐事繁多，跟人沟通不畅带来的痛苦。组长这个职位是我选择的，若这真不是我想要的，可以放弃。若是想挑战下，就应该正面迎上去，解决这些问题，而不是经常抱怨。当组长期间，经常碰到解决不了的问题时，便推向个大组长或者其他人，这应是自己承担责任的时候。碰到问题，承担它所带来的的痛苦，便成长了，便有了更多的选择，便更自由。逃避问题带来的痛苦，也就相当于逃避了自由。
