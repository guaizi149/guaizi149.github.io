        # 2019总结

        虽记得19年初时也是雄心壮志，在心里列出了一堆目标，可惜现在已全部忘记，只能凭借不那么灵光的脑袋大致的回顾一下了~ 先说工作吧。今年全面接手了理财的管理工作。总体的感觉用一句话概括就是力不从心。 以前总感觉自己交流以及管理方面的才能还是不错的，只是没有合适的机会展现出来。一年下来，也算是对自己有了些清醒的认识。不管是管理还是交流，都不是自己擅长的，只有代码还尚可慰藉。只是今年接手管理工作后，放在代码上的时间自然少了，技术提升上完全没有感觉，这才是比较让我心慌的。上半年主要是跟着产品的需求走，下半年接入了umi框架，其实还有部分尚待完善。代码能力提升太少，管理能力和交流能力有一点点提升。大概就是今年工作上的总结了。 再说生活吧。可说的东西也不多，唯一可安慰的就是看着宝宝一点点长大，这种感觉还是相当美妙的。期望可以一直呵护他，让他健康快乐的长大成人，可能也就是我余生的一大追求了。 明年的期望嘛。希望todo list上在年底能多几条划线吧。
